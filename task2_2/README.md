## Инструкция по запуску тестов
___

1. Склонируйте к себе репозиторий, в котором хранится проект тестового задания, через выполнение команды в терминале
```
git clone https://github.com/ArslanEfimov/avito-qa-internship-solution.git
```

2. Убедитесь, что на вашем компьютере установлен JDK 21+. В командной строке/терминале выполните команду

```
java -version
```

Если он не установлен, то установите с официального [сайта Oracle](https://www.oracle.com/java/technologies/downloads/) выбрав подходящую версию для вашей операционной системы, и пройдите шаг сначала.
Не забудьте потом добавить файл в переменную окружения

3. Через командную строку/терминал перейдите в корневую директорию задачи 2, выполнив команду

```
cd /{путь до директории проекта}/task2_2
```

4. Так как уже есть Gradle Wrapper, то сам gradle устанавливать не надо. Для запуска тестов осталось запустить следующую команду

```
./gradlew test
```
